Choice UI Layout

When a script hits Choice { options: [...] }, where should buttons appear?

A

Center Overlay

Choices appear as a centered modal overlay, darkening the background. Typical in many JRPG/console VNs. Blocks everything until a choice is made.

B

Bottom Area (above dialogue box)

Choices appear just above the dialogue box, replacing the main content area. The dialogue box stays visible. Found in many PC VNs (e.g., Fate, Muv-Luv).

C

Replace Dialogue Box

The dialogue box itself transforms into a choice list. Each option looks like an extended dialogue line. Least intrusive, very text-focused.
