状態

S

Save モード

空スロットをクリック → 新規セーブ／埋まっているスロットをクリック → 上書き確認

L

Load モード

埋まっているスロットのみクリック可／空スロットはグレーアウト

F

ファイル I/O

SaveManager.serialize/deserialize → JSON ファイルを saves/ に保存

データフロー

Save:
  SaveSlot clicked → confirm dialog → snapshot ScriptEngine state
  → SaveManager::save_slot(idx) → serialize → writes saves/slot_{idx}.json

Load:
  SaveSlot clicked (filled) → confirm dialog → read saves/slot_{idx}.json
  → deserialize → restore ScriptEngine + AffectionMap → reload script → jump to saved line

Close:
  Escape → despawn save overlay → back to Menu state
